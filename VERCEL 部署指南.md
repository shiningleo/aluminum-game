## 📋 5 分钟部署步骤

### 步骤 1：访问 Vercel
打开浏览器，访问：**https://vercel.com**

### 步骤 2：登录/注册
- 点击 "Sign Up" 或 "Login"
- 使用 GitHub / GitLab / Bitbucket 账号登录
- 或直接使用邮箱注册

### 步骤 3：创建项目
1. 登录后点击 "**Add New Project**"
2. 选择 "**Deploy Regular Project**"（不是 Git 仓库）

### 步骤 4：上传文件
**方法 A：拖拽上传**
- 将 `aluminum-game` 文件夹整个拖入浏览器窗口

**方法 B：选择文件夹**
- 点击 "Browse" 选择 `C:\Users\Administrator\.easyclaw\workspace\aluminum-game`

### 步骤 5：部署
1. 项目名称：输入 `aluminum-game`（或自定义）
2. 点击 "**Deploy**"
3. 等待约 30 秒...

### 步骤 6：获得链接！
部署完成后，您会获得永久链接：
```
https://aluminum-game-xxx.vercel.app
```
