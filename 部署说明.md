# 🚀 铝业大亨 - 快速部署指南

## ✅ 本地服务器已启动！

**局域网访问地址**：
```
http://10.2.0.56:8080
```

---

## 📱 方案一：同一 WiFi 下手机访问（推荐）

1. 确保手机和电脑连接**同一 WiFi**
2. 手机浏览器打开：`http://10.2.0.56:8080`
3. 即可游戏！

---

## 🌐 方案二：内网穿透（公网访问）

### 使用 ngrok（推荐）

1. 下载 ngrok：https://ngrok.com/download
2. 解压后运行：
   ```bash
   ngrok http 8080
   ```
3. 获得公网链接（如：`https://abc123.ngrok.io`）
4. 分享链接给任何人！

### 使用 Cloudflare Tunnel（免费稳定）

1. 访问：https://try.cloudflare.com/
2. 自动获得临时隧道链接

---

## ☁️ 方案三：手动部署到 Vercel（永久链接）

1. 访问 https://vercel.com
2. 注册/登录 GitHub 账号
3. 点击 "Add New Project"
4. 将 `aluminum-game` 文件夹拖入浏览器
5. 自动部署，获得永久链接！

---

## 📧 方案四：发送文件

直接发送 `index.html` 文件，对方用浏览器打开即可离线游玩！

---

## 🛑 停止服务器

按 `Ctrl+C` 或在命令行输入：
```bash
# 找到进程 ID 后
taskkill /PID <进程号> /F
```
