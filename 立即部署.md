## 📋 3 步完成部署（2 分钟）

### 第 1 步：登录 Vercel
在打开的页面中：
- 点击 **"Continue with GitHub"**（推荐，最快）
- 或用邮箱登录

### 第 2 步：导入项目
1. 登录后，在 **"Import Project"** 页面
2. 点击 **"Add New Project"**
3. 选择 **"Deploy Regular Project"**
4. 在弹出窗口中，**拖入整个文件夹**：
   ```
   C:\Users\Administrator\.easyclaw\workspace\aluminum-game
   ```
   或点击 **"Browse"** 选择该文件夹

### 第 3 步：部署
1. 项目名称：`aluminum-game`
2. 点击 **"Deploy"**
3. 等待 30-60 秒...
4. ✅ 完成！获得链接如：
   ```
   https://aluminum-game-xxx.vercel.app
   ```

---
